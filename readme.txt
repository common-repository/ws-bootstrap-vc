== Description ==
WS Bootstrap connect Wordpress and Bootstrap for you, create shortcodes with bootstrap elements and add new elements in WP Bakery so you can add it everywhere on your website

List of Bootstrap elements:
- Buttons
- Cards
- Jumbotrons
- Modals
- Popovers
- Sliders

== Installation ==
MANUAL INSTALLATION
The manual installation method involves downloading our plugin and uploading it to your webserver via your favorite FTP application.

Download the plugin file to your computer and unzip it
Using an FTP program, or your hosting control panel, upload the unzipped plugin folder to your WordPress installations wp-content/plugins/ directory.
Activate the plugin from the Plugins menu within the WordPress admin.

OR:
From your WordPress dashboard

Visit Plugins > Add New
Search for “WS Bootstrap”
Install the “WS Bootstrap” plugin
Activate “WS Bootstrap” from your Plugins page
